# Intro_GIS

Materials for 2026 Cascadia R Conference workshop, "Intro to GIS and Mapping in R"

Before the workshop, please make sure to install R, Rstudio, and the most recent versions of the following R packages:   
- here  
- dplyr  
- lubridate  
- ggplot2  
- ggthemes  
- httr  
- patchwork  
- viridis  
- knitr  
- terra  
- sf  
- tidyterra  
- leaflet  
- leafem  
- htmltools  
- units
